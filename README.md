# Steaph API

## Requirements

* Start up mongo server
```
$ mongod
```

## Target environemnt

* Start up Steaph API
```
$ npm install
$ npm run tsc
$ npm start
```

## Development environment

* Start up TypeScript Compiler
```
$ npm install
$ npm run tscw
```

* Start up Seaph API
```
$ nodemon
```

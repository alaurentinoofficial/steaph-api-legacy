# Steaph API

## Requirements

* Mongo v2.6.10 in :27017
* Node.js v6.5

## Target environemnt

* Start up Steaph API
```
$ npm install
$ npm run tsc
$ npm start
```

## Development environment

* Start up TypeScript Compiler
```
$ npm install
$ npm run tscw
```

* Start up Seaph API
```
$ nodemon
```
